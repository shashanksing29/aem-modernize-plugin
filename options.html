How Detection Works

The extension uses the AEM Modernize Tools native endpoints — no manual path configuration needed:

Component scan: .component.pagedata.json?path=<page> — returns which components have matching conversion rules.

Template status: .template.rules.json?path=<page> — if rules are returned the page uses a legacy static template; if empty it's already using an editable template.

Rule fetching: .component.rules.json and .template.rules.json — fetch the exact rules to include in conversion job payloads.

Credentials are stored locally in Chrome storage and only sent to your configured AEM server.
